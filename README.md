# Emotion Prediction Web App

A **Django-based web application** for predicting human emotions using **two models**:  

1. A traditional machine learning model 
2. A neural network model  

Users can interact with the app via a web interface to test both models in real time.

---

## Features

- **Dual Models:** Compare predictions from a classic ML model and a neural network model.  
- **Django Web Interface:** Upload input text and view predictions.  
- **Real-time Feedback:** Immediate prediction results from both models with accuracy score. 
- **Model Evaluation:** Backend includes metrics like Accuracy, Precision, Recall, F1-Score.  
- **User-friendly:** Designed so anyone can try the models through the browser.

---

## Tech Stack

- Python 3.x  
- Django
- Data Analysis: pandas, NumPy 
- Machine Learning: scikit-learn (traditional ML) 
- Neural Network: TensorFlow / KeraS
- Frontend: HTML / CSS
